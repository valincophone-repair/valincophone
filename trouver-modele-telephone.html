Guide pratique pour identifier un iPhone ou un smartphone Android

Conseils proposés par Valinco Phone à Propriano

Avant de demander un devis ou de commander une pièce, il est important d’identifier précisément le modèle de son téléphone. Ce guide vous aide à retrouver le modèle exact de votre iPhone, Samsung Galaxy, Xiaomi, Huawei ou autre smartphone Android.

Pourquoi c’est important ?
Le modèle exact de votre téléphone détermine le prix de la réparation, le type de pièce à commander et le délai d’intervention. Une erreur de modèle peut entraîner un coût différent, une mauvaise pièce et un retard de 24 à 48 heures ou plus.

1. Identifier le modèle dans les réglages

iPhone

  • Ouvrez Réglages
  • Allez dans Général
  • Puis Informations
  • Repérez le nom du modèle et le numéro du modèle

Exemple : iPhone 11, iPhone 12 Pro, iPhone 13 Pro Max, avec un numéro du type A2633 ou A2221.

Android : Samsung, Xiaomi, Huawei, Oppo, Redmi, Poco…

  • Ouvrez Paramètres
  • Allez dans À propos du téléphone
  • Notez le nom de l’appareil et le numéro du modèle

Le numéro du modèle peut ressembler à : SM-xxxx, Mxxxx, CPHxxxx, VOG-L29, 2201117TY, etc.

Attention : des modèles très proches peuvent être totalement différents. Par exemple : Galaxy A52, A52s et A52 5G ne sont pas le même appareil.

2. Trouver le modèle grâce à l’IMEI

L’IMEI permet souvent de confirmer précisément l’identité du téléphone.

  • Composez *#06# depuis le téléphone
  • L’IMEI s’affiche à l’écran

L’IMEI peut aussi se trouver :

  • sur la boîte d’origine
  • sur la facture d’achat
  • dans les réglages du téléphone
  • parfois sur le tiroir SIM

3. Vérifier la boîte d’origine ou la facture

La boîte et la facture sont souvent les sources les plus fiables après les réglages.

  • La boîte indique souvent le modèle exact
  • La facture peut préciser le modèle, la capacité et parfois la couleur
  • L’espace client de votre opérateur peut aussi afficher le bon appareil

4. Regarder les marquages physiques

Selon le modèle, certaines informations peuvent apparaître :

  • à l’arrière du téléphone
  • sur le tiroir SIM
  • sous la batterie pour certains anciens modèles

Cette méthode est moins pratique, mais peut aider en cas de téléphone bloqué ou cassé.

5. Vérifier les comptes en ligne

Dans certains cas, votre compte en ligne permet de retrouver le téléphone enregistré.

  • Compte Apple / iCloud
  • Compte Google
  • Compte Samsung
  • Compte Xiaomi
  • Espace client opérateur

6. Exemples de modèles souvent confondus

  • iPhone 11 / iPhone 11 Pro / iPhone 11 Pro Max
  • iPhone 12 / iPhone 12 Pro
  • Samsung Galaxy A52 / A52s / A52 5G
  • Samsung Galaxy S21 / S21 FE / S21 Ultra
  • Xiaomi Redmi Note 11 / Note 11S / Note 11 Pro
  • Huawei P30 / P30 Pro / P30 Lite
Une simple confusion de nom peut suffire à commander le mauvais écran, la mauvaise batterie ou la mauvaise caméra.
En cas d’erreur de modèle :

• Le tarif initial peut ne plus être valable si le modèle réel diffère du modèle annoncé.
• Une nouvelle pièce peut devoir être commandée.
• Le délai peut être prolongé de 24 à 48 heures, voire davantage selon le transport ou les jours fériés.

Pour éviter cela, il est toujours préférable de vérifier soigneusement le modèle avant tout devis ou paiement.

7. Règle importante

Ne validez jamais un devis ou un paiement sans vérifier le modèle exact de votre téléphone.

Besoin d’aide ?
Vous pouvez envoyer une photo des réglages, du dos du téléphone, du tiroir SIM ou de la boîte afin que le modèle soit vérifié.
Besoin d’un devis ou d’un conseil ?
Une fois le modèle identifié, vous pouvez utiliser la page Demande de devis vous pouvez aussi revenir sur la page d’accueil de Valinco Phone afin de mieux connaître le fonctionnement de l’atelier à Propriano.
